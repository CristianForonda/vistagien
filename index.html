ANÁLISIS DE DATOS ECOLÓGICOS Y AMBIENTALES

Resumen

Se presenta a continuación, los insumos requeridos para el análisis de datos ambientales y ecológicos mediante códigos del entorno R, abordando diferentes aspectos de la bioestadística y el análisis multivariado. Esta información es el resultado de un proyecto soportado por el grupo de investigación GIEN, orientado a la generación de un espacio web en el que se puedan efectuar procedimientos para el análisis de datos ecológicos y ambientales.

Presentacion

El análisis de datos en herramientas de distribución gratuita ha sido una tendencia en las últimas décadas, dada las facilidades que ofrece a usuarios, en especial para países en vía de desarrollo. Es por ello, que a continuación se ofrecen herramientas de entorno web y un repositorio de comandos orientados al análisis de datos ecológicos y ambientales, iniciando con la exploración de patrones gráficos, siguiendo con las transformaciones y estandarizaciones y finalizando con diferentes técnicas multivariadas y el análisis de diversidad taxonómica. Esta información será complementado por un texto que contextualiza a las diferentes temáticas.
